Detailed Description

This class provides access to the Enobio device and its data streaming.

This class provides methods for opening the device, starting and stopping the data streaming and configuring the device.

It inheritates from the base class IDataProcessor so it provides producer implementations to which IDataConsumer implementations can register to access both streaming data and device status.

bool Enobio::openDevice (  ) 

It performs the required operations to open and initialize the hardware device

Returns:
It returns true if the device is opened correctly, false otherwise. When the device is not present or can not be opened an error status is also passed to the IDataConsumer implementations already registered to the Enobio::STATUS producer.
void Enobio::closeDevice (  ) 

It performs the required operations to close the hardware device.

void Enobio::startStreaming (  ) 

It starts the Enobio data sampling. The data streaming will then be received at the IDataConsumer implementations already registered to the Enobio::ENOBIO_DATA producer. If the Enobio device is not turned on or it is out of battery a time-out will occur and the corresponding error status is generated by the Enobio::STATUS producer.

void Enobio::stopStreaming (  ) 

It stops the Enobio data sampling. After this call some samples might still be provided by the Enobio::ENOBIO_DATA producer due to the latency in the wireless link.
